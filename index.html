About SWARG

SWARG Shield is a premium encryption tool that protects your sensitive data with military-grade AES-256-GCM encryption. All encryption happens locally on your device - no data ever leaves your browser.

Security Features

  • Real AES-256-GCM: Military grade encryption using Web Crypto API
  • PBKDF2 Key Derivation: 100,000 iterations for strong key generation
  • Zero Knowledge: All encryption happens locally - no data sent to servers
  • Self-Destructing Tokens: Tokens auto-destroy after expiry time
  • Memory Security: Sensitive data wiped from memory after use
  • Clipboard Protection: Auto-clear clipboard after 30 seconds

How to Use

  • Encrypt: Enter password → Type text → Click Encrypt → Copy token
  • Share: Share token with recipient (via secure channel)
  • Decrypt: Enter same password → Paste token → Click Decrypt
  • Important: Share password and token via different channels
